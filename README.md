# 🎮 The World of Mohit

![Mohit's Portfolio](image/Profile.jpeg)

Welcome to **The World of Mohit**, a cinematic, high-fidelity portfolio showcasing my journey as a **Game Developer & Creator**. 

## 🚀 Experience the Journey
This isn't just a portfolio—it's an interactive experience built with **HTML5, CSS3, and JavaScript**, featuring GSAP animations and 3D elements to bring game development concepts to life.

### 🌟 Key Features
- **Cinematic Environment**: A dark, torch-lit "Ancient Temple" aesthetic.
- **Interactive UI**: Custom HUD-style technical skill meters and game-inspired navigation.
- **3D Integration**: Seamless parallax effects and smooth transitions.
- **Responsive Design**: Optimized for all devices—from mobile to ultra-wide monitors.

## 🛠 Tech Stack
- **Engine**: Unity 3D, C#
- **Web**: HTML, CSS (Vanilla), JavaScript, GSAP
- **Tools**: VS Code, Blender, Python

## 🕹 Featured Projects
